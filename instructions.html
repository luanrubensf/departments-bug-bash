🏗️ How It Works

A facility can now have multiple departments. Each department has its own isolated set of settings:

Positions & Qualifications Time Bins & Timeslot Presets Schedule Configuration Staffing Targets Schedule & Master Schedule

When a new department is created, default time bins and schedule configuration are generated automatically. Everything else must be set up by users or Maple admins.

🧪 How to Test
  • 01 The feature flag is already enabled for the entire dev environment — no action needed to enable it.
  • 02 Set up departments for your facility using a facility or Maple admin. Navigate to Settings > Departments.
  • 03 Configure the department: create positions, qualifications, timeslots, etc. (same flow as setting up a new facility).
  • 04 For units, a single unit can be linked to multiple departments. Access the unit page as a Maple admin to link it.
  • 05 To create a Department Admin: go to the Roster, add a new employee, select the "Department Admin" role, and link them to departments.
  • 06 Employee–department links are driven by active qualifications. Workers with qualifications across departments are visible to all relevant departments.
